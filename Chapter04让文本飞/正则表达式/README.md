# 正则表达式

## 位置标记
+ `^`: 例`^tux`匹配以`tux`起始的行
+ `$`: 例`tux$`匹配以`tux`结尾的行

## 标识符
+ A字符: 匹配该字符A
+ `.`: 匹配任意字符
+ `[]`: 匹配中括号内的任意一个字符
+ `[^]`: 匹配不在中括号内的任意一个字符

## 数量修饰符
+ `?`: 匹配之前的项1次或0次
+ `+`: 匹配之前的项1次或多次
+ `*`: 匹配之前的项0次或多次
+ `{n}`: 匹配之前的项n次
+ `{n,}`: 之前的项至少需要匹配n次
+ `{n,m}`: 之前的项所必须匹配的最小次数的最大次数

## 其他类型
+ `()`: 将括号中视为一个整体
+ `|`: 匹配两边任意一项
+ `\`: 转义字符可以转义之前介绍的特殊字符

## 匹配任意单词
`( +[a-zA-Z]+ +)`: 第一个`+`和最后一个`+`表示匹配一个或多个空格

## 匹配IP地址
`[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}`: 这个正则表达式不检查合法性
